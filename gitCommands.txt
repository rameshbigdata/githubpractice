First Step after installing the git:
===================================

setting your git credentials:
-----------------------------
First, set your username:

git config --global user.name "Your Name"


Then, set your email:

git config --global user.email youremail@example.com


Validate your configuration was saved:

git config --list
You should see your `user.name` and `user.email` values listed 


Setup Vs code as a default editor:
==================================
Open the config file with defailt editor:
git config --global e 

To change the editor to VS Code:

git config --global core.editor code

Try again opening the config file:

git config --global e 


Setup VSCode as the default DiffTool
============================================

Set and verify the difftool variable in our global config

git config --global diff.tool code
git config --global --get diff.tool

Set the actual execution command for code as the difftool, and verify:

git config --global difftool.code.cmd 'code --wait --diff $LOCAL $REMOTE'

git config --global --get difftool.code.cmd

Set VSCode as the Default Merge Tool:
=====================================
Add the entry for the merge tool
git config --global merge.tool code
git config --global mergetool.code.cmd "code --wait $MERGED"


Add settings to avoid prompting and keeping backups
git config --global mergetool.prompt false
git config --global mergetool.keepbackup false

Review the config settings:
git config --global --list
